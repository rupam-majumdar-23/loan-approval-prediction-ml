# Loan Approval Prediction using Machine Learning

This project predicts whether a loan application will be approved using machine learning algorithms.

## Project Overview
The model analyzes applicant details such as income, credit score, loan amount, years of employment, and internal bank points to determine loan approval.

## Algorithms Used
- Decision Tree
- Random Forest

## Deployment
The trained model is deployed using Streamlit to build an interactive web application.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Streamlit
