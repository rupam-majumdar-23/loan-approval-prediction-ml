# 🚀 Loan Approval Prediction using Machine Learning


## 📌 Project Overview

This project predicts whether a **loan application will be approved or rejected** using Machine Learning algorithms.

The model analyzes applicant information such as income, credit score, loan amount, years of employment, and bank points to make predictions.

## 🧠 Algorithms Used

✅ Decision Tree Classifier  
✅ Random Forest Classifier  

Random Forest was used as the final prediction model due to its strong performance.



## 📊 Project Workflow

1️⃣ Data Loading

3️⃣ Exploratory Data Analysis (EDA)  
4️⃣ Model Training
